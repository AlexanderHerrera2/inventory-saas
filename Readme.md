Proyecto SaaS para multiples empresas.
Este proyecto se desarrolló con el objetivo de mejorar mi portafolio y aprender mas conceptos
necesarios de programación.
Se optó por usar React.js para frontend, Node.js para backend junto con prisma y para la base
de datos se decidió usar PostgreSQL. Esto con el objetivo de familiciarze mas con el mercado
laboral actual y las verdaderas necesidades en el entorno laboral.
